# ericfarewell.com

Static site for Eric Farewell — somatic business coaching. Watercolor + Peacock design system.
Built and maintained with Claude. Auto-deploys to Cloudflare Pages on push to `main`.

- Pages: index, my-story, coaching, programs, speaking, podcasts, blog, wall-of-love, contact
- Shared styles: assets/css/tokens.css (design tokens) + assets/css/site.css
- Shared JS: assets/js/site.js
- No build step. Output = repo root.
